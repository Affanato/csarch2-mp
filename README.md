## Setup

1. **Initialize Virtual Environment**

```console
> py -3 -m venv .venv
```

2. **Open Virtual Environment**

```console
> .venv\Scripts\activate
```

3. **Install dependencies in Virtual Environment**

```console
$ pip install Flask
```

4. **Start Server**

```console
$ flask run
```
